# survey-server
Open Source Survey Tool generated by ChatGPT-4

## Prerequisites

- JDK 1.8 or later
- Maven 3.2 or later
- An IDE like IntelliJ or Eclipse

## Installation

1. Clone the repository:
```shell
git clone https://github.com/your-displayName/survey-server.git
```
2. Import the project into your favorite IDE.

## Running the Application

To run the application, execute the following command in the terminal:
```shell
mvn spring-boot:run
```

The application will start running at `http://localhost:8080`.

## API spec
http://localhost:8080/api/v1/swagger-ui/


## Testing

You can use Postman to test the API endpoints. Import the Postman collections provided in the previous sections to get started quickly.
